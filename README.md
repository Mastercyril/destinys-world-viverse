# destinys-world-viverse

Destiny's World - Lost Society in Time. A complete 3D investigation game for VIVERSE with 4 zones, temporal fragments, and multiplayer support.

- Feature #1-5: Complete (Player, AlienAI, Dialogue, NPC, Quest systems)
- Feature #6: Stealth and Sensory System (StealthSystem.js) - IN PROGRESS
- Feature #7: Save/Load System - PENDING
- Feature #8: Achievement System - PENDING

- Perplexity AI integration
- VIVERSE/PlayCanvas 3D engine
